# 源码文件说明

本目录存放期末研究报告《基于异构并行的遗传算法求解 TSP：多模型对比与混合优化研究》的全部源代码。
项目围绕旅行商问题（TSP），实现了 8 类求解算法（NN、NN+2-opt、SA、Serial-GA、OpenMP-MS、OpenMP-Island、CUDA-Base、CUDA-Memetic）及若干实验脚本。

---

## 目录总览

```
代码/
├── common/          # 公共模块：数据结构、TSPLIB 解析、遗传算子、2-opt、随机数、计时器
├── serial/          # 串行基准算法：NN、NN+2-opt、SA、Serial-GA
├── openmp/          # OpenMP 并行 GA：主从模型、岛屿模型
├── cuda/            # CUDA 并行 GA：Base、Memetic、Top-k Memetic
└── scripts/         # 实验与绘图脚本（benchmark、敏感性、收敛、可扩展性等）
```

---

## 一、common/ — 公共模块

| 文件 | 作用 |
|---|---|
| `common.h` | 定义统一数据结构：`Problem`（问题实例）、`Individual`（个体/染色体）、`GAParams`（GA 参数）、`GAResult`（运行结果）；内联实现距离查询、路径评估、最优个体查找等工具函数。 |
| `tsplib.c` / `tsplib.h` | 解析 TSPLIB 标准算例（当前支持 `EUC_2D` 边权类型），计算并存储城市间距离矩阵。 |
| `ga_ops.c` / `ga_ops.h` | 遗传算法核心算子：Fisher-Yates 随机初始化、锦标赛选择、顺序交叉（OX）、交换变异、种群评估等。 |
| `two_opt.c` / `two_opt.h` | 经典 2-opt 局部搜索实现，用于 NN+2-opt、SA、CUDA-Memetic 等算法。 |
| `random.h` | 64-bit 线性同余伪随机数生成器（LCG），支持每线程独立 RNG 状态，避免多线程/多核竞争。 |
| `timer.h` | 跨平台高精度计时器封装：Windows 使用 `QueryPerformanceCounter`，Unix 使用 `gettimeofday`。 |

## 二、serial/ — 串行基准算法

| 文件 | 作用 |
|---|---|
| `serial_baseline.c` | 串行算法统一入口程序，通过命令行参数选择运行 `nn`、`2opt`、`sa` 或 `ga` 模式。 |
| `nn.c` / `nn.h` | 最近邻法（Nearest Neighbor）构造启发式实现。 |
| `sa.c` / `sa.h` | 模拟退火（Simulated Annealing）实现：以随机 2-opt 移动为邻域操作，按 Metropolis 准则接受劣解。 |
| `ga_serial.c` / `ga_serial.h` | 串行遗传算法实现，作为所有并行 GA 的正确性与性能基准。 |
| `Makefile` | 编译 `serial_baseline.exe` 的 Makefile。 |

---

## 三、openmp/ — OpenMP 并行遗传算法

| 文件 | 作用 |
|---|---|
| `main.c` | OpenMP 程序入口，支持 `ms`（主从模型）和 `island`（岛屿模型）两种模式，统一解析命令行参数并输出结果。 |
| `ga_openmp_ms.c` / `ga_openmp_ms.h` | OpenMP 主从模型 GA：每代将种群个体静态划分给各线程并行评估/交叉/变异，主线程负责全局精英保留。 |
| `ga_openmp_island.c` / `ga_openmp_island.h` | OpenMP 岛屿模型 GA：将种群划分为若干子种群，每个线程独立演化一个岛屿，并按环状拓扑定期迁移最优个体。迁移函数保证 `mig_size` 个个体替换到 `mig_size` 个不同的最差位置，并保护前 `elite_count` 个精英。 |
| `Makefile` | 编译 `ga_openmp.exe` 的 Makefile，含 `-fopenmp` 选项。 |

---

## 四、cuda/ — CUDA GPU 并行遗传算法

| 文件 | 作用 |
|---|---|
| `main.cu` | CUDA 程序入口，支持三种运行模式：`base`（基础细粒度 GA）、`memetic`（周期性 CPU 2-opt 精炼）、`memetic-topk`（Top-k 个体精炼）。 |
| `cuda_common.cu` / `cuda_common.cuh` | CUDA 公共模块：设备内存分配/释放、距离矩阵按行拷贝到 GPU、Host 端 Top-k 索引查找、CUDA 错误检查宏等。 |
| `cuda_kernels.cu` / `cuda_kernels.cuh` | CUDA 核心核函数：种群初始化、适应度评估、选择/交叉/变异进化核函数。 |
| `ga_cuda_base.cu` / `ga_cuda_base.cuh` | CUDA-Base：纯 GPU 细粒度 GA，一线程一个体，精英保留由 Host 端完成。 |
| `ga_cuda_memetic.cu` / `ga_cuda_memetic.cuh` | CUDA-Memetic：GPU GA + CPU 2-opt 混合算法。每隔固定代数将当前最优个体传回 CPU 执行 2-opt 精炼，再传回 GPU。 |
| `ga_cuda_memetic_topk.cu` / `ga_cuda_memetic_topk.cuh` | Top-k Memetic 变体：每次精炼前 k 个最优个体，而非仅 Top-1。 |
| `Makefile` / `build.bat` | CUDA 编译脚本。`build.bat` 会先将源码复制到临时 ASCII 路径再调用 NVCC，以规避中文路径问题。 |

## 五、scripts/ — 实验与可视化脚本

| 文件 | 作用 |
|---|---|
| `benchmark.py` | 小中规模统一 benchmark：运行全部 8 类算法，覆盖多个 TSPLIB 算例与随机种子，结果写入 `results/benchmark.csv`。 |
| `benchmark_large.py` | 大规模 benchmark：仅运行 NN+2-opt、SA、CUDA-Memetic，结果写入 `results/benchmark_large.csv`。 |
| `scalability.py` | 可扩展性实验：测试 OpenMP 线程数与 CUDA 种群规模对运行时间的影响，结果写入 `results/scalability.csv`。 |
| `generate_convergence_berlin52.py` | 生成 berlin52 实例上 Serial-GA、CUDA-Base、CUDA-Memetic 的收敛曲线 CSV。 |
| `generate_convergence_a280.py` | 生成 a280 实例上 CUDA-Base、CUDA-Memetic 的收敛曲线 CSV。 |
| `convergence_multi.py` | 生成多个规模实例（berlin52、eil76、ch150、a280）上 CUDA-Base 与 CUDA-Memetic 的对比收敛曲线 CSV（写入 `results/fig/`）。 |
| `plot_convergence.py` | 绘制 berlin52 收敛曲线图。 |
| `plot_convergence_a280.py` | 绘制 a280 收敛曲线图。 |
| `plot_convergence_multi.py` | 绘制跨规模收敛对比子图。 |
| `interval_sensitivity.py` | 2-opt 精炼间隔敏感性实验（τ ∈ {10,25,50,100,200}），结果写入 `results/interval_sensitivity.csv`。 |
| `param_sensitivity.py` | 种群规模与迭代代数敏感性实验，结果写入 `results/sensitivity.csv`。 |
| `topk_experiment.py` | Top-k 精炼强度实验（k ∈ {1,3,5,10}），结果写入 `results/topk_experiment.csv`。 |
| `time_to_quality.py` | SA 与 CUDA-Memetic 的时间-质量权衡曲线，结果写入 `results/time_to_quality.csv`。 |
| `plot_*.py` | 对应上述实验的绘图脚本，生成 `results/fig/` 下的 PNG 图表。 |
| `generate_report_tables.py` | 从 `results/*.csv` 自动生成 LaTeX 报告中的 `table_small.tex`、`table_large.tex`、`table_parallel.tex`、`table_topk.tex`。 |
| `update_openmp_island.py` | （维护脚本）在修复 OpenMP-Island 迁移 bug 后，仅重新运行并更新 `benchmark.csv` 中的 OpenMP-Island 结果。 |
| `download_tsplib.py` | 自动下载 TSPLIB 标准算例（已下载完毕，通常无需再运行）。 |
| `gen_tsp.py` | 生成随机 TSP 实例（本研究主要使用 TSPLIB，该脚本用于辅助测试）。 |

## 七、构建与复现说明

1. **编译全部可执行文件**
   - Windows：在项目根目录运行 `build_all.bat`
   - 或分别进入 `code/serial`、`code/openmp`、`code/cuda` 执行 `make`（Windows 下使用 `mingw32-make`）

2. **一键复现实验**
   - 在项目根目录运行 `python reproduce.py`（会自动编译并运行所有实验）
   - 若已编译，可运行 `python reproduce.py --skip-build` 跳过编译步骤

3. **依赖**
   - C 编译器：GCC 10+（支持 OpenMP）
   - CUDA：NVCC 13+（用于 CUDA 代码）
   - Python：3.10+，依赖 pandas、matplotlib、numpy（项目自带 `plot_env` 虚拟环境）

## 八、注意事项

- 本目录中不再包含 `.o`、`.exe`、`__pycache__/` 等编译/运行产物；需要时可按上述命令重新生成。
- CUDA 代码默认使用 `CUDA_PATH=C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v13.3` 与 `ARCH=sm_120`，若本地 CUDA 版本或 GPU 架构不同，请在 `cuda/build.bat` 或环境变量中调整。
- 所有随机算法均支持通过命令行参数固定随机种子，以保证结果可复现。
